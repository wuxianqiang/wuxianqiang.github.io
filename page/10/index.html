shell中的变量

Posted on 2019-07-04

什么是变量？

  1. 可以变化的量
  2. 变量必须以字母或下划线开头，名字中间只能由字母，数字和下划线组成
  3. 变量名的长度不得超过255个字符
  4. 变量名在有效范围内必须唯一
  5. 变量默认类型都是字符串

Posted on 2019-07-03

WebSocket 与传统 HTTP 有如下好处。

  1. 客户端与服务器端只建立一个 TCP 连接，可以使用更少的连接。
  2. WebSocket 服务器端可以推送数据到客户端，这远比 HTTP 请求响应模式更灵活、更高效。
